# 江苏第二师范学院2014大创项目——翻转课堂教学平台(全栈开发) #

## 项目技术
    1. 后台
        + 后台开发语言NodeJS(v0.10)
        + 后台开发框架express(v4)
        + 模块引擎jade

    2. 前台
        + 前台开发框架Bootstrap(v3)
        + HTML5、CSS3、javascript、ajax

    3. 数据库
        + mongodb(v2.6)
        + ORM:mongoose(v3.8)
